Tree Type Exploration

Beyond the classic layered evergreen — what PNW tree silhouettes work inside a pin?

Layered Fir (C6)

Classic PNW evergreen. Safe, recognizable.

Sitka Spruce

Taller, more tiers. Coastal PNW feel. Detailed.

Western Red Cedar

Drooping tips suggest cedar. Distinctive but complex.

Lodgepole Pine

Tall trunk, compact canopy. Very different silhouette.

Campsite Fir

Canopy up top, exposed trunk. What you see at camp.

Madrona (attempt)

Organic curve + round canopy. Beautiful tree, tricky as an icon.

Two Pines

Forest scene — two different-height trees. More depth.

Tent + Tree

Campsite scene — tree + tent. Tells the whole story.
