How it works:
1. User clicks the "Dates" field → calendar pops down
2. Click a start date → it highlights as the start
3. Hover shows preview range → click end date → range fills in
4. Click "Done" or click outside to close
5. Quick presets (This weekend, Jun, etc.) auto-set the range and close the calendar
6. "Nights" stays as a separate field — in "Find a date" mode it's editable, in "Exact dates" mode it auto-calculates from the range
